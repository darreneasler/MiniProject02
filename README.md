# MiniProject02

This python code takes data from the "Telco" telecom company's sales data and create a miultivariable linear regression model beased on the monthly services the company offers to predict the final monthly bill of customers based on which of the services they subscribe to. Based on this linear regression model, the services and their respective monthly charges are as follows. 


Service 1: phone_service for $20.05 per month.
Service 2: multiple_lines for $5.02 per month.
Service 3: online_security for $5.05 per month.
Service 4: online_backup for $4.99 per month.
Service 5: device_protection for $5.01 per month.
Service 6: tech_support for $5.02 per month.
Service 7: streaming_tv for $9.98 per month.
Service 8: streaming_movies for $9.95 per month.
Service 9: internet_fo for $24.94 per month.
Service 10: internet_no for $-25.06 per month. 

The data boasts very good errors metrics, an MAE of $0.80, an RMSE of $1.06, and and R_2 of 0.99874.
Each Jupyter Notebook cell is labeled with its function.
